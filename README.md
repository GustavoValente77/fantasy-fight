# Fantasy Fight

Fullstack web application inspired by fantasy sports, focused on fight predictions.

## About the Project

Fantasy Fight is a study project developed to simulate a fight fantasy experience.  
Users can select fighters and make predictions based on match outcomes.

This project was built as part of my portfolio to practice fullstack development, including frontend, backend, and API integration.

## Technologies

- Angular
- Node.js
- Express
- MongoDB

## Features

- User authentication (login/register)
- Fighter selection interface
- Betting/prediction system
- Dynamic UI interactions

## ⚠️ Important Notes

- This project uses a limited dataset of fighters for demonstration purposes
- It is not connected to real-time fight results
- Some data was collected for educational use only
- Built for study and portfolio purposes
